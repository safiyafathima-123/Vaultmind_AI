# 🚀 PL Genesis AI: Autonomous Multi-Agent Multi-Chain Strategist

**PL Genesis AI** is a next-generation DeFi execution engine that solves the "Black Box AI" problem. Built for the **PL Genesis** hackathon, it provides verifiable, autonomous yield optimization by anchoring every decision with a dual-proof transparency layer on **IPFS** and **NEAR Protocol**.

---

## 📽️ The Pitch: "Transparent Autonomy"
Most AI agents trade in secret—you don't know why they make trades, and you can't prove their data hasn't been tampered with. **PL Genesis AI** is different. It uses a 3-agent debate engine (*Yield Hunter*, *Risk Guardian*, and *Stability AI*) to reach consensus, then proves its "Thinking" and "Actions" on-chain.

---

## 🏗️ Integrated Sponsor Ecosystem

### 💎 Protocol Labs (IPFS/Filecoin) - "Thinking Transparency"
The AI doesn't just execute; it debates. We hash the full 3-agent debate logs and generate an **IPFS CIDv1** for every single decision. 
- **Proof of Integrity:** You can audit the mathematical CID directly in the dashboard using our "Verify Integrity" modal.
- **Permanent Web:** Reasoning logs are prepared for permanent archival on the Filecoin network.

### 🌐 NEAR Protocol - "Execution Transparency"
Every strategy change is anchored with a live **NEAR Protocol Testnet** transaction.
- **On-Chain Proof:** Users get immediate, clickable NearBlocks receipts for every action the AI takes.
- **Accountable Agents:** The AI signs transactions on behalf of the user's session, creating a verifiable paper trail of financial accountability.

---

## ✨ Key Features
- **3-Agent Debate Engine:** Real-time consensus between Yield (Profit), Risk (Safety), and Stability (Sentiment) agents.
- **Verify Integrity Modal:** A built-in cryptographic audit tool that proves your logic logs match the IPFS CIDv1 signature.
- **Voltage Radar:** A 4-axis visualizer scoring pool safety, APY, confidence, and depth.
- **Frictionless Experience:** No wallet extension required for the demo—just enter your NEAR Testnet ID and the AI handles the execution proof.

## 🛠 Tech Stack
- **Dashboard:** Next.js (App Router), Tailwind CSS (Aesthetic Dark Mode), Recharts.
- **Strategy Engine:** Node.js, `near-api-js`, IPFS-CID-v1 Hashing.
- **Blockchain:** NEAR Protocol Testnet.
- **Storage:** IPFS (Reasoning Logs).

## 🚀 Presentation Setup

### 1. Backend AI Engine
```bash
cd backend
# Create .env from .env.example (Add your NEAR Testnet credentials)
npm install
npm run dev
```

### 2. Frontend Dashboard
```bash
npm install
npm run dev
```
Visit **http://localhost:3000** and enter your NEAR ID (e.g., `safiya2.testnet`) to begin the autonomous sequence.
